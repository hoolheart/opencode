## Platform Visual Guidelines

### Bilibili Video

**Format Specifications:**
- Resolution: 1920x1080 (1080p) or higher
- Aspect ratio: 16:9
- Frame rate: 30fps or 60fps
- Duration: 5-15 minutes (optimal for most content)

**Visual Style Elements:**
- Clean, professional aesthetic
- Logical visual progression
- Text overlays for emphasis (Chinese subtitles important)
- Consistent branding elements (intro/outro, lower thirds)
- Thumbnail design: Clear focal point, readable text, high contrast

**Content Structure:**
```
0:00-0:10: Hook (visual + audio grab attention)
0:10-0:30: Introduction (who, what, why)
0:30-[main]: Core content (visual variety every 30-60 seconds)
[last 1 min]: Summary and CTA
```

**INTP-A-H Adaptations:**
- Avoid overly energetic, hype-driven editing
- Prefer clean transitions over flashy effects
- Substantive visual information over pure entertainment
- Authentic presentation style

### Xiaohongshu Images

**Format Specifications:**
- Aspect ratios: 3:4 (portrait), 1:1 (square), 9:16 (vertical video)
- Resolution: Minimum 1080px on shortest side
- Format: JPG for photos, PNG for graphics with transparency

**Visual Style Elements:**
- High-quality, well-lit photography
- Aesthetic consistency with brand visual direction
- Natural, lifestyle-oriented compositions
- Strategic negative space for text overlays
- Color palette aligned with brand

**Image Types:**
- Lifestyle photos: Authentic, relatable scenes
- Flat lays: Organized, aesthetic product/content displays
- Infographics: Information presented visually
- Quote cards: Text-based graphics with brand fonts
- Before/after: Transformation or comparison content

**INTP-A-H Adaptations:**
- Clean, uncluttered compositions
- Intellectual aesthetic (books, workspace, thoughtful scenes)
- Authentic rather than overly styled
- Substance over superficial beauty

### Xiaohongshu Short Videos

**Format Specifications:**
- Duration: 15-60 seconds
- Aspect ratio: 9:16 (vertical)
- Resolution: 1080x1920
- Captions: Recommended (many users watch without sound)

**Visual Style Elements:**
- Quick hook in first 3 seconds
- Fast-paced but clear visual storytelling
- Text overlays for key points
- Trending audio integration
- Clean transitions

**INTP-A-H Adaptations:**
- Educational or insightful content over pure entertainment
- Thoughtful pacing (not hyperactive)
- Substantive value in short format

### WeChat Graphics

**Format Specifications:**
- Cover image: 900x383px (2.35:1 ratio)
- Inline images: Width 640-1080px, flexible height
- File size: Optimized for fast mobile loading

**Visual Style Elements:**
- Professional, clean design
- Mobile-optimized readability
- Brand color consistency
- Clear visual hierarchy
- Strategic use of white space

**Graphic Types:**
- Header images: Article introduction visuals
- Section dividers: Visual breaks between content sections
- Infographics: Data and process visualization
- Quote highlights: Key message emphasis
- CTA buttons: Clear call-to-action graphics

**INTP-A-H Adaptations:**
- Minimalist, uncluttered designs
- Intellectual aesthetic
- Substantive visual information
- Avoid overly promotional or salesy graphics
